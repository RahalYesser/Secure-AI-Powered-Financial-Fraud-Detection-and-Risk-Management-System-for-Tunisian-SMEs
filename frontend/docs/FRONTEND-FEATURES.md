# Frontend Features & Pages

This document focuses on the main screens and features implemented in the React frontend.

## Dashboard

- Consolidated KPIs for transactions, fraud patterns, credit risk, and users.
- Charts for transaction trends and credit-risk distribution.

## Transactions Page

- Paginated table of transactions.
- Filtering and detail views.

## Fraud Detection Page

- Manual fraud detection form by transaction ID.
- Detection result view with ensemble and per-model predictions.
- Fraud patterns table with pattern type, description, confidence-derived severity, and review status.

## Credit Risk Page

- List of credit-risk assessments with filters (by category, unreviewed-only).
- New assessment modal to submit SME financial data.
- Assessment result and detailed view with risk score, category, summary, and model predictions.

## User Management & Profile

- User management for admins/auditors: list users, lock/unlock accounts.
- User profile page: basic profile and role information.
